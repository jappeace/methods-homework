\subsection{Where can cont not be pushed up?}
If the Q is being split up on both sides of the proof, for example:
\begin{prooftree}

\AxiomC{$
	\Gamma, Q \vdash A \times B
$}
\AxiomC{$
	\Delta, Q, A, B \vdash C
$}
\RightLabel{$E\times$}
\BinaryInfC{$
	\Gamma, \Delta, Q, Q \vdash C
$}
\RightLabel{Cont}
\UnaryInfC{$
	\Gamma, \Delta, Q \vdash C
$}
\end{prooftree}

Now cont can't be pushed up because one side would have nothing to cont.
