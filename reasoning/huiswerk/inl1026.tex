Kripke models: If $I(\Gamma \Rightarrow A)$ holds in all Kripke models, then
so does $I(\Pi, \Box \Gamma \Rightarrow \Box A, \Delta)$

\begin{prooftree}
	\AxiomC{$\Gamma \Rightarrow A$}
	\RightLabel{$R_k$}
